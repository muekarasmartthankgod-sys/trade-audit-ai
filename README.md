# Global Trade Audit AI 🚢

An AI-powered compliance engine designed to help exporters and importers identify risks in their shipping documentation before submission.

### How to use:
1. Upload a Shipping PDF (Invoice, BoL, or Packing List).
2. The AI extracts 10+ key data points.
3. The system runs a cross-reference check against global trade standards.

Test the demo here: https://trade-audit-ai-kqefxmftigyq4lectascj2.streamlit.app/
